# Battleship Game in C

This is a console-based Battleship game developed in C.

## Features
- Two-player gameplay
- Ship placement with validation
- Hit / Miss system
- Ship destruction tracking
- Game loop until one player wins

## Technologies
- C Programming Language

## Description
The game simulates the classic Battleship game where players place ships on a 20x20 grid and take turns attacking each other's board.

This project was developed as part of my training in software development.
